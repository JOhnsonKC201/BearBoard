==============================================================================
  GENERAL NOTES
==============================================================================

  - Backend runs with: cd backend && uvicorn main:app --reload
  - Frontend runs with: cd frontend && npm install && npm run dev
  - Backend API docs: http://localhost:8000/docs (Swagger UI)
  - Frontend dev server: http://localhost:3000

  - The backend uses SQLAlchemy + FastAPI. The frontend uses React 18 +
    Vite + Tailwind CSS + React Router.

  - The backend endpoints are all built and functional. The frontend UI
    matches the mockup design. The main remaining work is connecting the
    frontend to the backend API (auth context, API calls, live data).

  - Coordinate on API contracts before building. Agree on request/response
    shapes in the team chat before coding endpoints.

  - Every feature branch needs a PR. Every PR needs at least one review.

  API ENDPOINTS (all implemented):
    POST   /api/auth/register     - Register new user
    POST   /api/auth/login        - Login, returns JWT (24h expiry)
    GET    /api/auth/me           - Current user from token
    GET    /api/users/{id}        - Get user profile
    PUT    /api/users/{id}        - Update own profile (auth required)
    GET    /api/posts             - List posts (?sort=&category=&limit=&offset=)
    POST   /api/posts             - Create post (auth required)
    GET    /api/posts/{id}        - Single post with comments
    DELETE /api/posts/{id}        - Delete own post (auth required)
    POST   /api/posts/{id}/vote   - Vote on post (auth required)
    GET    /api/trending          - Top 3 trending posts
    GET    /api/events            - Upcoming events
    GET    /api/groups            - Study groups
    POST   /api/chat              - Chat assistant (canned responses)
